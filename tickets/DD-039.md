# DD-039

## Title

Fix active socket use for add, list, and collector.

## Method

TDD

## Problem

`ensure_agent` can select or start a live socket that differs from the default managed socket path. `ssh.add`, loaded-key checks, and list mode were still calling `ssh-add` against the default path, which can fail with `Error connecting to agent` even after a live socket has been selected.

## Acceptance

- `ssh.add` runs `ssh-add` against the active socket, not the default managed path
- collector and list mode read `ssh-add -l` from the active socket
- returned `agent` fields report the active socket
- Docker tests and coverage pass
- installed DD command proof passes
- latest DD source command proof passes
- release commit and push complete
