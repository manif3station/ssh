# DD-038

## Title

Add `ssh.list` and `ssh.ls`.

## Method

TDD

## Problem

Users can register keys with `dashboard ssh.add`, but they need a quick way to inspect the skill registry and see whether each remembered key is loaded in ssh-agent without opening the config file manually.

## Acceptance

- `dashboard ssh.list` prints a table by default
- `dashboard ssh.ls` is an alias of `dashboard ssh.list`
- `-o table` and `-o json` are supported
- output includes registry key, expanded file path, loaded status, and fingerprint when available
- list mode reports `loaded`, `not-loaded`, and `missing-file`
- Docker tests and coverage pass
- installed DD command proof passes
- latest DD source command proof passes
- release commit and push complete
